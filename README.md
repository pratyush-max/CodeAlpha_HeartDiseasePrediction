## 📌 Overview

This project uses **Machine Learning algorithms** to predict whether a person is likely to have heart disease based on medical attributes.
It includes **data analysis, visualization, model building, and evaluation**.

---

## 🧠 Models Used

✨ Logistic Regression
🌳 Decision Tree
🌲 Random Forest (**Best Model**)
